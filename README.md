# 📱 프롬프터 (리모콘 조작형)

폰을 **프롬프터 화면**으로, 다른 기기(노트북·태블릿·다른 폰)를 **리모콘**으로 쓰는 웹앱.
같은 와이파이(또는 폰 핫스팟)에 있으면 브라우저 주소만 열면 됩니다. 설치·앱스토어·서명 없음.

## 실행

```bash
cd prompter
npm install      # 최초 1회
npm start
```

서버가 뜨면 터미널에 접속 주소가 표시됩니다. 노트북 브라우저에서 시작 페이지를 열면
프롬프터/리모콘 주소와 **QR 코드**가 나옵니다.

- 시작 페이지: `http://<노트북IP>:4173/`
- 프롬프터(폰):  `http://<노트북IP>:4173/display`
- 리모콘:        `http://<노트북IP>:4173/remote`

> 포트는 기본 4173. 바꾸려면 `PORT=5000 npm start`.

## 쓰는 법

1. **폰**에서 `/display` QR을 찍어 프롬프터 화면을 연다.
2. **조작 기기**에서 `/remote` QR을 찍어 리모콘을 연다.
3. 리모콘의 **"대본 편집 / 불러오기"** 를 펼쳐 대본을 붙여넣거나 **📄 파일 불러오기**로
   `.txt`(또는 `.md`, `.srt`, `.vtt`) 파일을 올린다 → 불러오면 곧바로 프롬프터에 표시됨.
   직접 입력/수정 후엔 **"프롬프터에 보내기"**.
4. **▶︎ 재생** 으로 자동 스크롤 시작. 속도·글자크기·위치를 실시간 조작.

## 리모콘 기능

- 재생/정지, 처음으로(⏮)·끝으로(⏭)
- 진행바 드래그로 위치 이동
- ▲▼ 미세 이동(조금 위/아래)
- 속도 ± (px/초), 글자크기 ±
- 기준선 표시, 좌우반전·상하반전(텔레프롬프터 유리용)
- 대본 편집/전송, **파일 업로드(.txt/.md/.srt/.vtt)** — 자막 파일은 타임코드·번호 자동 제거
- 연결 상태 표시(끊기면 자동 재연결)

## 구조

- `server.js` — 정적 서빙 + WebSocket 릴레이(상태 캐시 → 늦게 접속한 기기도 자동 동기화)
- `public/display.html` — 프롬프터 화면(자체 애니메이션 엔진)
- `public/remote.html` — 리모콘
- `public/index.html` — 시작 페이지(주소 + QR)

## 프롬프터 화면에서 직접 조작

리모콘 없이 프롬프터 화면(폰)에서도 위치를 맞출 수 있습니다.

- **위아래로 쓸기(스와이프)** — 텍스트 위치 이동(위로 쓸면 전진)
- **화면 우하단 ▲▼ 버튼** — 누르고 있으면 계속 이동
- **짧게 탭** — 재생/정지
- (노트북) **마우스 휠** = 위/아래, **스페이스** = 재생/정지, **방향키** = 미세 이동

수동 조작하면 자동 스크롤은 자동으로 멈춥니다.
